Privacy and Notice

Local data, unofficial status, and rulebook boundaries.

Character data stays in this browser unless you export JSON. There is no account, backend sync, analytics, or remote storage in the current app.

This is an unofficial fan tool and portfolio project. It is not affiliated with, endorsed by, sponsored by, or approved by Pinnacle Entertainment Group.

Use the official books for rules. The app should support table workflow without replacing the source material.

Campaign Profile

Sources & Rulesets

This app currently assumes a Deadlands: The Weird West campaign using the listed books. Use the listed sources for character options unless the app marks an option as unsupported, forbidden, or not yet implemented.

Profile

  • Browning Private Security & Detective Agency
  • Deadlands: The Weird West focused profile

Books Used

  • Savage Worlds: Adventure Edition
  • Deadlands: The Weird West
  • Deadlands Weird West Companion

Rules Assumptions

  • Encumbrance is used
  • Minimum Strength is used
  • Wealth is tracked
  • Starting Wealth: $250
  • Starting Attribute Points: 5
  • Starting Skill Points: 12
  • Extra Perk Points: 0

Character Creation Scope

  • Human is the only supported ancestry or race in the first implementation slice
  • Core skills include Athletics, Common Knowledge, Notice, Persuasion, and Stealth
  • Electronics and Hacking are not used for this profile
  • Trade based on Smarts is part of the campaign profile
  • Gear, Powers, Vehicles, Allies, and full campaign customization are later slices

Current Limitation

  • This page is informational for now
  • The app does not currently support changing source books or campaign settings
  • Future versions may make this profile data-driven or editable
